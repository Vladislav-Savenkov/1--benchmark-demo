Классификатор алкогольной продукции ЕГАИС

Классификатор предназначен для хранения сведений об алкогольной продукции ЕГАИС. Классификатор ведется на федеральном уровне. Новые элементы добавляются в классификатор при загрузке документов из ЕГАИС, при запросе алкогольной продукции по коду ФСРАР или по ИНН производителя (импортера).

Элементы классификатора должны быть сопоставлены с номенклатурой предприятия. Одной позиции классификатора можно сопоставить несколько элементов номенклатуры. В поле Сопоставлено в ИБ отображается гиперссылка, текст которой зависит от количества сопоставленных элементов справочника номенклатуры:

  • Не сопоставлено, если нет ни одного сопоставленного элемента справочника номенклатуры.
  • Название сопоставленной номенклатуры, если сопоставлена только одна позиция.
  • Несколько позиций (X), с указанием конкретного количества позиций.


В форме списка есть возможность запрашивать алкогольную продукцию по считанной акцизной с помощью сканера штрихкодов.
