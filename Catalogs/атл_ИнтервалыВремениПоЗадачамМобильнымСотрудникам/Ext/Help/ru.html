Интервалы времени по задачам

Справочник предназначен для хранения интервалов времени по задачам, которые планируется анализировать при помощи отчета "Затраты времени на выполнение заявок".

Под интервалом времени понимается время, находящееся в диапазоне между наступлением двух событий. Допустим, мы хотим проанализировать время, которое сотрудник тратит непосредственно на решение задачи, т.е. мы должны анализировать время, между наступлением двух событий: приездом сотрудника по указанному адресу и фактическим завершением работ.

Для этого необходимо создать интервал, в котором в качестве имени реквизита даты начала указать "ПриехалДата", а в качестве имени реквизита даты конца "ЗавершилДата". При этом имена реквизитов даты начала и даты конца должны в точности соответствовать именам реквизитов документа "Задача мобильному сотруднику".

Для упрощения наполнения справочника в верхней части формы списка предусмотрена кнопка "Заполнить по умолчанию", при нажатии на которую справочник заполняется автоматически наиболее часто используемыми интервалами.

Если наполнение (или дополнение) справочника происходит после начала эксплуатации системы, т.е. в системе уже есть введенные документы "Задача мобильному сотруднику", после ввода новых интервалов необходимо воспользоваться кнопкой "Пересчитать в задачах", чтобы в расчетах был учтен вновь созданный интервал. Перерасчет выполняется при помощи обработки "Пересчет времени перехода в статус по задачам".
