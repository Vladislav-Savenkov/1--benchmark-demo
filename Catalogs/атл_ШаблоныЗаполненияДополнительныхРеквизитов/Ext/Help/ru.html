Справочник "Шаблоны заполнения дополнительных реквизитов"

Справочник предназначен для хранения шаблонов заполнения дополнительных реквизитов, которые задаются в задаче мобильному сотруднику. Использование шаблонов позволяет оптимизировать трудозатраты на подготовку данных по задачам мобильному сотруднику. 

Алгоритм заполнения значений дополнительных реквизиты описывается на встроенном языке "1С:Предприятие 8". Шаблон срабатывает в момент заполнения документа "Задача мобильному сотруднику" по шаблону. В шаблоне можно использовать следующие параметры:

  • Форма - форма документа "Задача мобильному сотруднику";
  • ЗадачаИсполнителя - ссылка на документ "Задача исполнителя";
  • ДокументОснование - документ занятости ресурса, который указывается в документе "Задача мобильному сотруднику";
  • Результат - значение дополнительного реквизита, которое будет присвоено по выполнению шаблона.
