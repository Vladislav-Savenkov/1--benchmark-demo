Правила задания шаблонов заполнения дополнительных реквизитов

Алгоритм заполнения значений дополнительных реквизиты описывается на встроенном языке "1С:Предприятие 8".  Используется в момент заполнения документа "Задача мобильному сотруднику" по шаблону. В шаблоне возможно использовать следующие параметры:

  • Объект - объект документа "Задача мобильному сотруднику";
  • ЗадачаИсполнителя - ссылка на документ "Задача исполнителя";
  • ДокументОснование - документ занятости ресурса, который указывается в документе "Задача мобильному сотруднику";
  • Результат - значение дополнительного реквизита, которое будет присвоено по выполнению шаблона.
