Справочник "Шаблоны заполнения задач мобильным сотрудникам"

Справочник предназначен для хранения шаблонов заполнения задач мобильным сотрудникам. В шаблоне указывается список дополнительных реквизитов, которые при выборе шаблона в задаче попадают в табличную часть "Дополнительные реквизиты". Реквизиты могут быть сгруппированы по произвольному признаку. Для каждого реквизита задается, на какой из сторон заполняется реквизит (1С или МК), обязательность заполнения реквизита на мобильном устройстве, значение реквизита по умолчанию, шаблон заполнения реквизита (алгоритм по которому заполняется реквизит).
