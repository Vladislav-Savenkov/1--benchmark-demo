Волгоградская область

Обработка предназначена для выгрузки сведений о начисленных суммах за жилое помещение и коммунальные услуги гражданам, имеющим право на меры социальной поддержки, в отдел социальной защиты населения Волгоградской области за указанный период.
 
Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Период» - месяц, за который необходимо предоставить данные;
  • «Файл загрузки» - путь к файлу формата *.DBF, из которого будет загружена необходимая информация для заполнения табличной части обработки;
  • «Кодировка файла» - кодировка файла загрузки и выгрузки в формате *.DBF, которое может принимать значения DOS и Windows;
  • «Каталог выгрузки» - каталог выгружаемого файла в формате *.DBF.
 
Перед началом выполнения обработки необходимо произвести её настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «Настройка» на форме обработки.
В открывшемся окне необходимо произвести настройку соответствия услуг в ОСЗН и элементов справочника «Услуги» в базе.

Для загрузки данных из файла и заполнение данными из базы необходимо нажать на кнопку "Заполнить все". Если структура файла не будет соответствовать структуре, описанной в таблице, то загрузка данных будет невозможна.
Поиск данных из файла и БД происходит по следующим полям:
  • SURNAME
  • NAME

  • SECNAME

  • TOWN

  • STREET
  • KORPUS
  • FLAT
  • HOUSE
Для выгрузки данных из табличных частей обработки в файлы, каталог размещения которых указан в поле «Каталог выгрузки», необходимо нажать на кнопку «Выгрузить все».
Формат файла выгрузки таблицы начислений соответствует формату файла загрузки. Имя выгружаемого файла «uszn3.dbf».
