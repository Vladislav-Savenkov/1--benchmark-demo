Ярославская область

Обработка предназначена для выгрузки данных в отдел социальной защиты населения Ярославской области за указанный период.

Перед началом выгрузки данных в обработке необходимо заполнить следующие параметры:
  • «Организация» - ссылка на элемент справочника «Организации», по которой будут браться необходимые данные за указанный период;
  • «Месяц выгрузки» - месяц, за который необходимо предоставить данные;  
  • «Вариант выгрузки» - вариант выгрузки данных
  • «Каталог выгрузки» - указывается каталог для выгружаемого файла в формате *.DBF;
 
Параметр «Вариант выгрузки» может принимать два значения:
  • Сведения о начисленных льготах;
  • Сведения о начислениях по услугам.

В зависимости от выбранного значения варианта выгрузки структура файла выгрузки будет различаться.

Перед началом заполнения/выгрузки данных, для корректного заполнения табличных частей обработки, необходимо произвести предварительную настройку. Для вызова диалогового окна настройки необходимо нажать на кнопку «1. Настройка».
1. В табличной части вкладки «Соответствия услуг» необходимо установить соответствия между услугами в ОСЗН и услугами в организации. Для этого необходимо указать коды услуг в соответствующих полях.
2. В табличной части вкладки «Соответствия льготных категорий» необходимо установить соответствия между льготными категориями в ОСЗН и льготными категориями в организации. Для этого необходимо указать коды льготных категорий в соответствующих полях. При этом один и тот же код льготной категории может установлен для нескольких льготных категорий, использующихся в программе.

Для заполнения табличной части обработки данными из БД необходимо нажать на кнопку «2. Заполнить». Для выгрузки данных из табличной части обработки в каталог указанный в поле «Каталог выгрузки» необходимо нажать на кнопку «3. Выгрузить».

Значение поля

PFR Страховой номер СНИЛС.
FM Фамилия.
IM Имя.
OT Отчество.
DTR Дата рождения.
INDX Почтовый индекс.

NSP

 Населенный пункт.
ULC Улица.
DOM Дом.
KORP Корпус.
KVAR Квартира.
KOMNATA Комната.
CAT_ID Категория льготника (код).
USLUGA Вид услуги (из справочника). На каждый вид услуги заполняется новая строка в файле. Количество строк равно количеству услуг, оказанных организацией по человеку.
SUMMA Сумма компенсации (сумма к выплате по данной услуге).
SUMMA_PER

Сумма перерасчета.

SUMMA_ALL Полная стоимость за услугу.
PR Пустое поле (не заполняется организацией).
ORG_ID

Организация сделавшая расчет по записи (справочник). Код организации указывается в наименовании файла. Только в том случае, если файл содержит данные от нескольких организаций можно показать в этом поле коды этих организаций, иначе оставить пустым.
