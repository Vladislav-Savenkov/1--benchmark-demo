Пересчет времени перехода в статус по задачам

Обработка предназначена для пересчета интервалов по задаче.  Использовать обработку целесообразно, когда в справочник "Интервалы времени по задачам" после начала эксплуатации системы добавляются новые интервалы.  Анализ данных по интервалам времени осуществляется при помощи отчета "Затраты времени на выполнение заявок".

Обработка может быть запущена из справочника "Интервалы времени по задачам".
