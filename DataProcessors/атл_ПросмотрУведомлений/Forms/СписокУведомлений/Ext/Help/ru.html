Список уведомлений

Форма списка уведомлений предназначена для информирования пользователей 1С о наступлении событий, происходящих с мобильными сотрудниками.

Перечень событий, информация о которых попадает в список:

  • Изменение статуса задачи ("Получил", "Принял", "Отказался", Выехал", "Приехал", "Завершил").
  • Новые комментарии от мобильного сотрудника по задачам.

Новые события выделяются в списке жирным шрифтом. После прочтения уведомления пользователь может установить отметку о прочтении (кнопка "Установить "прочитано" у выбранных уведомлений") при этом шрифт выделения изменится с жирного на обычный. Для прочтения уведомления в отдельном окне (форма уведомления по задаче) необходимо кликнуть по нему два раза. Такой способ, удобен для просмотра уведомлений о новых комментариях, где может быть длинный текст. Для того чтобы скрыть неактуальные уведомления из списка необходимо воспользоваться кнопкой "Скрыть выбранные уведомления".

Установка отметки о прочтении, а также скрытие уведомлений из списка может осуществляться как по одному уведомлению, так и по группе. Выбор группы уведомлений осуществляется курсором при удерживаемой клавише "Ctrl".

Список уведомлений обновляется автоматически, с периодичностью, строго заданной в конфигурации. Для принудительного обновления необходимо нажать кнопку "Обновить" (располагается в верхней части списка).

Список уведомлений может открываться при запуске конфигурации автоматически, для этого необходимо в настройках пользователя установить флаг "Показывать список уведомлений при запуске программы".
