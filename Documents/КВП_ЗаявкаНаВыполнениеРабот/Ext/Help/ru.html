Документ «Заявка на выполнение работ»

Документ предназначен для регистрации заявок и жалоб от жильцов, а также планирования и отражения факта выполнения работ по ремонту жилого фонда.

При вводе документа необходимо указать:

  • «Номер» – номер документа, устанавливается автоматически;
  • «Дата» – дата документа, устанавливается автоматически;
  • «Организация» – организация, к которой относится заявка, выбирается из справочника «Организации»;
  • «Вид операции» – вид операции документа, для которого доступны следующие виды операций: «Выполнение работ по лицевому счету», «Выполнение работ по дому»;
  • «Приоритет» – приоритет выполнения работ (высокий, средний, низкий);
  • «Категория работ» – категория выполняемых работ, для выбора которой доступны следующие категории: «Плановый ремонт», «Текущий ремонт», «Капитальный ремонт», «Аварийный ремонт»;
  • «Текст заявки» – текст или описание жалобы, поступившей от жильцов;
  • «Ответственный» – пользователь, ответственный за ввод и заполнение документа;

  • «Комментарий» – текстовый комментарий к документу.

Заполнение данных по заявке осуществляется на следующих страницах документа: «Основные сведения», «Работы и оборудование», «Наряд заказ», «Выполнение» и «Отражение в регл. учете».

Основные сведения

  • «Здание» – здание,  выбирается из справочника «Здания, сооружения»;
  • «Подъезд» – подъезд здания, выбирается из справочника «Подъезды»; 
  • «Помещение» – помещение здания; выбирается из справочника «Помещения»; поле доступно для вида операции «Выполнение работ по лицевому счету» заявки. При заполненном здании помещение выбирается из числа помещений этого здания, при заполненном подъезде - из числа помещений подъезда;
  • «Лицевой счет» – лицевой счет; выбирается из справочника «Лицевые счета»; поле доступно для вида операции «Выполнение работ по лицевому счету» заявки. Заполняется автоматически, если заполнено помещение, а в помещении открыт только один лицевой счет;
  • «Включать оборудование, установленное на подчиненные объекты» - признак доступен только для вида операции «Выполнение работ по дому». Если флаг установлен, то при подборе установленного на выбранное здание или подъезд оборудования будет также подбираться оборудование, установленное на подчиненные объекты учета: если заполнен подъезд, то на все помещения данного подъезда; если заполнено только здание, то на все подъезды и помещения данного здания.
  • «Код домофона» – код домофона подъезда, заполняется автоматически при выборе подъезда;
  • «Адрес» – адрес лицевого счета, заполняется автоматически при выборе лицевого счета;
  • «Контактное лицо» – заполняется автоматически при выборе лицевого счета, определяется по ответственному собственнику / нанимателю лицевого счета.
    • Если ответственный собственник / наниматель лицевого счета является жильцом, то заполняется данным жильцом.
    • Если ответственный собственник / наниматель лицевого счета является контрагентом, заполняется основным контактным лицом данного контрагента. Если основное контактное лицо контрагента не определено, то заполняется первым из списка контактных лиц данного контрагента.
  • «Телефон» – телефон, заполняется автоматически при выборе лицевого счета. Может быть отредактирован вручную. Принцип заполнения определяется по реквизиту «Контактное лицо»: 
    • Если «Контакное лицо» является жильцом, заполняется номером телефона связанного с жильцом физического лица. Если номеров телефона у физического лица несколько, становится доступен список выбора номера телефона.
    • Если «Контактное лицо» является контактным лицом контрагента, заполняется номером телефона контактного лица контрагента. Если номеров телефона у контактного лица несколько, становится доступен список выбора номера телефона. 
  • «Договор контрагента» - заполняется автоматически договором, участвующим во взаиморасчетах по выбранному лицевому счету.
  • «Дата и время, удобные для заказчика» – указывается дата и время, на которое запланировано выполнение заявки;
  • «Состав звена»:
    • «Ответственный»  – указывается мастер, ответственный за выполнение работ по текущей заявке. При заполнении ответственного мастера может производиться автоматическое заполнение типа цен отгрузки материалов по типу розничных цен оптового склада или склада с не указанным типом, ответственным за который является текущий мастер.
    • В табличной части указывается список мастеров, выполняющих работы по текущей заявке;
  • «Это повторное обращение» - флаг устанавливается в случае, если текущая заявка является повторным обращением. При установке флага из списка введенных ранее документов «Заявка на выполнение работ» выбирается первичное обращение.

Учет оборудования и работ 

На закладке «Наряд-заказ» расположены следующие таблицы: «Оборудование», «Признаки неисправности», «Работы».

Таблица «Оборудование»

В таблице задается список оборудования из справочника «Прочие объекты учета, оборудование». Доступно автоматическое заполнение списка оборудования всем оборудованием, сопоставимым с выбранными признаками неисправности и видами работ, а также подбор одного или нескольких значений сопоставимого оборудования. Определение сопоставимого оборудования осуществляется по следующим правилам:

  • Список оборудования ограничивается введенным в эксплуатацию оборудованием, установленным на объекте учета, по которому оформлена заявка. Если установлен флаг «Включать оборудование, установленное на подчиненные объекты», то в список оборудования включается оборудование, установленное на объекты учета, подчиненные объекту учета по заявке.
  • Если таблица «Признаки неисправности» заполнена, то список оборудования ограничивается оборудованием, к которому относятся признаки неисправности, указанные для вида текущего оборудования.
  • Если таблица «Работы» заполнена, то список оборудования ограничивается оборудованием, для которого доступны виды работ согласно данным регистра сведений «Сведения о видах работ, выполняемых с оборудованием».

Таблица «Признаки неисправности»

В таблице задается список признаков неисправности из справочника «Признаки неисправностей». Доступно автоматическое заполнение списка признаков неисправностей, сопоставимым с выбранным оборудованием и видами работ, а также подбор одного или нескольких значений сопоставимых признаков неисправности. Определение сопоставимых признаков неисправности осуществляется по следующим правилам:

  • Если таблица «Оборудование»,  заполнена, то список ограничивается признаками неисправности, относящимися к виду оборудования текущего оборудования.
  • Если таблица «Работы» заполнена, то список признаков неисправности ограничивается признаками неисправности, к которым относятся виды работ по данным табличной части «Виды работ» справочника «Признаки неисправностей».

Таблица «Работы» 

В таблице задается список видов работ из справочника «Виды работ». Доступно автоматическое заполнение списка видов работ, сопоставимым с выбранным оборудованием и признаками неисправности, а также подбор одного или нескольких значений сопоставимых видов работ. Определение сопоставимых видов работ осуществляется по следующим правилам:

  • Если таблица «Оборудование»,  заполнена, то список видов работ ограничивается видами работ, относящимися к оборудованию согласно данным регистра «Сведения о видах работ, выполняемых с оборудованием».
  • Если таблица «Признаки неисправности» заполнена, то список видов работ ограничивается видами работ, относящимся к признакам неисправности по данным табличной части «Виды работ» справочника «Признаки неисправностей».

Табличная часть «Работы» содержит следующие колонки:

  • «№» – порядковый номер строки;
  • «Вид работ» – указывается вид работ из справочника «Виды работ» ;
  • «Норма труда» – норма труда мастера, заполняется автоматически при выборе вида работ;
  • «Объем (план)» – указывается планируемое количество работ по заявке;
  • «Время (план)» – отображается планируемое на выполнение работы время. Рассчитывается автоматически исходя из нормы труда и планируемого объема работ. Может быть изменено вручную;
  • «Объем (факт)» – указывается количество выполненных работ по заявке (заполняется по факту);
  • «Время (факт)» – отображается фактическое время, затраченное на выполнение работы. Рассчитывается автоматически исходя из нормы труда и фактического объема работ. Может быть изменено вручную;
  • «Выполнена» – указывается признак выполнения вида работ. Если работа отмечена как выполненная, то по данному виду работ можно сформировать отгрузку услуг производственного характера с помощью документа «Оказание производственных услуг».

При нажатии кнопки «Заполнить факт по плану» автоматически во всех строках значение колонки «Объем (факт)» будет установлено равным значению колонки «Объем (план)».

При нажатии кнопки «Установить фактические объемы работ по времени выполнения заявки» фактические показатели времени, не заполненные вручную, будут автоматически заполнены пропорционально плановым показателям времени в пределах общего времени выполнения заявки. Общее время выполнения заявки определяется как разность времени окончания и начала работ, устанавливаемых на странице "Выполнение". Фактические показатели объема, не заполненные вручную, будут автоматически заполнены плановыми показателями объема работ.

Наряд - заказ

Табличная часть «Наряд-заказ» содержит следующие колонки:

  • «№» – порядковый номер строки;
  • «Вид работ» – из справочника «Виды работ» выбирается вид работ, по которой будет списан материал;
  • «Номенклатура» – выбирается материал из справочника «Номенклатура»;
  • «Вид материала» – ссылка на элемент справочника «Виды материалов» - указывает на вид материала, к которому относится номенклатура наряд-заказа;
  • «Количество (план)» – указывается количество материала, которое планируется потратить для выполнения данного вида работ;
  • «Количество (факт)» – указывается количество материала, которое потрачено при выполнении данного вида работ;
  • «Ед. измерения» – указывается ед.измерения из справочника «Классификатор единиц измерения»;
  • «Цена» – указывается цена номенклатуры;
  • «Ставка НДС» – указывается ставка НДС;
  • «Сумма НДС» – отображается сумма НДС;
  • «Сумма (план)» – отображается планируемая сумма материалов, которая необходима для выполнения данного вида работ;
  • «Сумма (факт)» – отображается фактическая сумма материалов, которая была потрачена при выполнении данного вида работ.

В командной панели табличной части расположена команда «Сформировать наряд-заказ» предназначенная для автоматического заполнения табличной части «Наряд заказ» по выбранным видам работ из табличной части «Работы». Заполнение выполняется по следующим правилам:

  • Если материалом в составе вида работ является ссылка на справочник «Номенклатура», то в строке таблицы «Наряд-заказ» в качестве номенклатуры указывается данный материал.
  • Если материалом в составе вида работ является ссылка на справочник «Виды материалов», то заполнение материала производится по следующим правилам. Для каждого оборудования из таблицы «Оборудование» и текущего вида метериалов определяется совместимый для оборудования материал по данным регистра сведений «Сведения о материалах, используемых при проведении работ с оборудованием».  
    • Если совместимый материал получен, и один, то он указывается в качестве используемого материала.
    • Если получено несколько совместимых с оборудованием материалов, то используемые материалы необходимо выбрать из списка совместимых материалов.
    • Если совместимых с оборудованием материалов нет, то используемые материалы необходимо выбрать из списка материалов, входящих в состав вида материала.

Примечание:

Уточнение используемых материалов в наряд-заказе производится в отдельной форме, доступной при выборе значения поля «Номенклатура».

Выполнение

На закладке «Выполнение» заполняются данных о результатах выполнения заявки:

  • «Статус выполнения» – устанавливается признак выполнения заявки;
  • «Оценка выполнения» – указывается оценка выполнения работ мастером;
  • «Дата и время начала работ» - указывается дата и время фактического начала работ по заявке;
  • «Дата и время окончания работ» - указывается дата и время фактического окончания работ по заявке. В случае, если заполена дата и время начала работ, оцениваются фактические трудозатраты по заявке.
  • «Замечания по выполнению» – указываются замечания по выполнению заявки.

Отражение в регл. учете

На данной закладке формируются типовые документы отражения списания материалов и отгрузки услуг по заявке, а именно :

  • «Реализация товаров и услуг» - при отгрузке материалов через реализацию формируется типовой документ «Реализация товаров и услуг». В качестве покупателя в документе отгрузки материалов по умолчанию указывается контрагент для взаиморасчетов по лицевому счету.
  • «Требование-накладная» - при списании материалов на оказание услуг производственного характера, выполненных в рамках заявки, формируется документ «Требование-накладная». Номенклатурные группы, на производство которых списываются материалы, по умолчанию определяются по номенклатуре выполненных видов работ или по материалам. Склад для списания материалов по умолчанию определяется по мастеру, ответственному за выполнение заявки, а именно, если мастер является ответственным за один склад с типом «Оптовый» или склад с не указанным типом склада, то данный склад автоматически указывается в требовании-накладной.
  • «Оказании производственных услуг» - при отражении выполненных по заявке работ как оказание услуг производственного характера формируются документы «Оказание производственных услуг». Формирование документов производится с группировкой по номенклатурным группам видов работ и только по тем видам работ, которые отмечены как выполненные.

Документ доступен в разделе «Аварийно-диспетчерская служба».
