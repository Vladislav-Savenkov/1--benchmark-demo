Услуги общедомового счетчика

Данный регистр предназначен для хранения услуг, объем которых будет учитываться при распределении показаний общедомовых приборов учета.
В регистре указываются поля:

  • «Услуга общедомового счетчика» - содержит ссылку на справочник «Услуги». Выбираются только услуги со способами расчета «По показаниям счетчика» или «По показаниям счетчика и норме потребления».
  • «Услуга-база расчета» - база расчета, содержит ссылку на справочник «Услуги». Способ расчета выбранной услуги должен совпадать со способом расчета услуги общедомового счетчика.
  • «Распределять на услугу-базу расчета» - если флажок установлен, то начисления по услуге и по услуге-базе расчета будут произведены одним документом «Начисление услуг». Распределение показаний общедомовых приборов учета также будет производиться на обе эти услуги.
  • «Коэффициент» - когда общедомовая услуга измеряется, например, в гигакалориях (гКал), а услуга-база расчета в кубических метрах (м3), необходимо объём потребления по услуге-базе расчета перевести из м3 в гКал и затем уменьшать расход по общедомовой услуге. В таком случае для перевода единиц измерения в данном поле необходимо указать количество услуги-базы расчета, содержащееся в единице услуги общедомового счетчика. По умолчанию заполнено единицей.

Услуга «Услуга общедомового счетчика» служит основанием для расчета услуги «Услуга», т.е. если производится расчет услуги «Услуга общедомового счетчика», то при расчете объема для распределения показаний общедомового прибора учета так же будет учтен объем потребленной услуги «Услуга» в расчетном периоде.

Важно!
Если начисление по услуге общедомового счетчика и услуге-базе расчета выполняется одним документом «Начисление услуг», а услуга-база расчета является зависимой, то начисление по такой услуге будет равно 0. Расчет будет производиться таким образом, потому что начисление зависимой услуги выполняется только при начислении соответствующей услуги-основания, указанной в регистре сведений «Зависимости услуг».

Для того, чтобы при расчете объема на общедомовые нужды учесть объемы потребления по услуге-базе расчета, являющейся зависимой услугой, необходимо перед начислением общедомовой услуги выполнить начисление услуги, являющейся основанием для услуги-базы расчета, отдельным документом «Начисление услуг». То есть, в данном случае начисление услуги общедомового счетчика и услуги-базы расчета нужно производить разными документами «Начисление услуг».

Регистр независимый, непериодический.
