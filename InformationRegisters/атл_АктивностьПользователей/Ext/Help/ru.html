Активность пользователей

Регистр сведений предназначен для хранения актуальных данных о последней активности пользователей. Данные хранятся как для мобильных сотрудников, так и для пользователей 1С.

Для мобильных сотрудников помимо времени последней активности хранятся данные о последнем местонахождении (широта и долгота).
