Затраченное время по интервалам по задачам мобильных сотрудников

Регистр предназначен для хранения количества затраченного времени в разрезе интервалов по задачам мобильных сотрудников. Пересчет регистра может быть выполнен при помощи обработки "Пересчет времени перехода в статус по задачам мобильным сотрудникам".
