Форма записи настроек подключения мобильных сотрудников

Предназначена для ввода и редактирования настроек подключения мобильных сотрудников к системе с мобильных устройств (через мобильное приложение). С помощью формы можно задать:

  • Логин - логин, с помощью которого мобильный сотрудник может авторизоваться с мобильного устройства. Перед авторизацией информация о логине должна быть передана на сервер обмена. Передача данных на сервер осуществляется с помощью обработки "Обмен данными с УМС".
  • Пароль - пароль, с помощью которого мобильный сотрудник может авторизоваться с мобильного устройства. Перед авторизацией информация о пароле должна быть передана на сервер обмена. Передача данных на сервер осуществляется с помощью обработки "Обмен данными с УМС". Пароль на сервере хранится в зашифрованном виде.
  • Приоритет - параметр, с помощью которого задается приоритет пользователя. Данные о пользователях с более высоким приоритетом выгружаются на сервер обмена в первую очередь. Наличие данных на сервере обмена является обязательным условием для авторизации сотрудника с мобильного устройства.

Рассмотрим пример для понимания параметра "Приоритет". Допустим компания использует

  • Флаг "Является пользователем МК"
  • Флаг "Подготовка к отключению"
  • Ключ шифрования
