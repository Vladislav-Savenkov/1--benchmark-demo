Настройки подсистемы "Управление мобильными сотрудниками"

Данный регистр сведений предназначен для хранения основных настроек подсистемы "Управление мобильными сотрудниками". Данные располагаются на трех закладках.

Закладка "Подключение к серверу"

  • Логин организации - уникальный логин компании строго идентифицирующий одну компанию, необходимый для подключения к серверу обмена. Логин можно получить в компании ООО "Мобильные решения для бизнеса" по e-mail sale@mobiforce.ru.
  • Пароль организации - пароль к логину организации, необходимый для подключения к серверу обмена. Пароль можно получить в компании ООО "Мобильные решения для бизнеса" по e-mail sale@mobiforce.ru.
  • Адрес сервера - интернет адрес сервера обмена данными. По умолчанию сервер находится по адресу: exchange.mobiforce.ru.
  • Периодичность синхронизации с сервером (сек.) - периодичность с которой конфигурация 1С производит обмен с сервером обмена. Минимальное возможное значение 15 секунд.
  • Периодичность запроса активности мобильных пользователей - периодичность с которой конфигурация 1С осуществляет запрос активности мобильного сотрудника. Статус "Активен" присваивается сотрудникам у которых на мобильном устройстве запущено приложение и обмен осуществляется в штатном режиме. Актуальные данные об активности сотрудников сохраняются в регистре сведений "Активность пользователей".
  • Пользователь для выполнения обмена данными - указывается пользователь, от имени которого будет запускаться обмен в фоновом режиме автоматически. Для запуска обмена в автоматическом режиме достаточно авторизоваться (войти) в 1С под данным пользователем.

Закладка "Шифрование"

  • Флаг "Использовать шифрование данных" - устанавливается, в случае если есть необходимость шифровать пакеты с данными при обмене с сервером. В качестве алгоритма шифрования используется симметричный алгоритм AES в режиме CBC с ключом 256 бит.
  • Общий ключ шифрования - ключ, который используется для шифрования пакетов с данными при обмене. Данный ключ всегда используется для шифрования справочных данных (если установлен флаг "Использовать шифрование данных", и может быть использован для шифрования адресных пакетов если другое не определено в настройках мобильных сотрудников.  

Закладка "Прочие настройки"

  • Адрес карты для отображения точек - интернет адрес страницы на которой выводится карта с указанием мест. Необходимо указывать для просмотра любой информации
