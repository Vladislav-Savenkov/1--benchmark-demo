Добавить новый справочник "ВидыОборудования" и обработку "ЗагрузкаВидовОборудования" для импорта данных из внешнего файла.

**Описание задачи:**

1. Создать новый справочник "ВидыОборудования" с реквизитами:
    - Код (Строка, длина 10)
    - Наименование (Строка, длина 100)
    - Описание (Строка, длина 255)
2. Создать обработку "ЗагрузкаВидовОборудования" (DataProcessors), которая позволяет выбрать внешний файл (например, CSV) и загрузить данные в справочник "ВидыОборудования".
3. В обработке реализовать модуль (bsl-файл) с процедурой "ЗагрузитьВидыОборудования", которая:
    - Открывает диалог выбора файла.
    - Читает данные из файла.
    - Добавляет новые элементы в справочник "ВидыОборудования" или обновляет существующие по коду.
Весь синтаксис пиши на русском языке, в лучших стандартах написания кода 1С
